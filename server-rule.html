サーバールール


これはどんなサーバーですか？

　　nC2(Twitter:@nc2_nicotube)によって作られた雑談用コミュニティサーバーです。
クリエイター同士や、クリエイターとフォロワーをより密接に繋げたいという意志の元作られました。

禁止事項はなんですか？また、罰則はどの様なものですか？

禁止事項

    　
  • 過度に悪口を言う、誹謗中傷する
    • 　
  • 何度も同じ書き込みをしたり、大量のURLや画像を書き込む
    • 　
  • 仲間外れや差別などをする
    • 　
  • その他モラルを守らない行為

罰則

本サーバーでは3ストライク制を採用しています。
罰則を受ける度に刑が重くなり、3回目で一時的にBANとなります。
  • 一回目:警告、1時間から8時間のタイムアウト(書き込み規制など)
  • 二回目:権限を引き下げ、1から4週間のタイムアウトもしくはkick
  • 三回目:権限をビジターに固定、1ヶ月から数ヶ月のBAN
  • 四回目:永久BAN
    • 永久BANになった場合、基本的にBAN解除は認められません。

サーバーのその他のルールはありますか？

  • サーバーに入ったらまず自己紹介をしてください。
    　　(名前orニックネーム、趣味…など)
    • 　
  • 専用のチャンネルにはそれに沿った内容を書き込んでください。
  • このサーバーを気に入ったのであれば是非このサーバーのことを共有してください。許可等はもちろん必要ありません。
